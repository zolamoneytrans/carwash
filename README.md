# carwash

This repository contains the CarWash web project.

## Notes
- The current workspace appears to be a PHP web application, not a native iOS app.
- A `codemagic.yaml` file has been added as a starting point for publishing an iOS app with bundle ID `africa.carwashes.noq`.
- You must update the `codemagic.yaml` build commands to match your actual iOS project structure before using Codemagic.
